# How to write Xulang code?

> [!NOTE]
> This tutorial is being continuously updated. Stay tuned, and contributions are welcome.

In Xulang's programming world, there are no variables and no functions. Everything is implemented through symbols, pattern matching, and substitution.

In Xulang, the following three parts are particularly important for programming:
1. Xulang basic syntax;
2. Xulang preprocessor commands;
3. Xulang standard library;

Even if you know nothing about Xulang's preprocessor commands and standard library, you can still use Xulang to write any algorithm you want to implement—though the code may be somewhat verbose or complex, which means Xulang is a programming language that requires little memorization of syntax rules.
